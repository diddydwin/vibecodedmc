# vibecodedmc
A Minecraft clone rewritten in JavaScript, made using Gemini's Pro models.

Run the "vibecodedmc.html" to play.

Like eaglercraft, it will ask you to create an account name, the default is "Player". (no mutliplayer, yet!)

Most buttons are there to make the game complete, they don't do much.

Some do work though.

Controls: 

W - Move foward
A - Move left
S - Moonwalk (move backward)
D - move right

Space - jump

GUI Controls:

T - Open chat

Commands:

/gamemode [survival/creative] - Toggles gamemode (only survival + creative)
